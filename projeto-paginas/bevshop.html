BevShop

Problema: Necessidade de um sistema de e-commerce funcional e organizado.

Solução: Desenvolvimento de uma aplicação web com integração entre front-end e banco de dados, simulando uma loja virtual real.

Tecnologias: HTML, CSS, JavaScript, Banco de Dados.

Meu papel: Participação na construção da interface, organização do sistema e estruturação das funcionalidades do projeto.

Resultado: Sistema funcional de e-commerce, preparado para receber produtos e expandir funcionalidades futuras.
